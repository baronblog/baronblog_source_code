Welcome to [Baron World](https://baronblog.github.io/)! This is my first post.

## Quick Start

### Create a new post

``` bash
$ hexo new "My New Post"
```


### Run server

``` bash
$ hexo server
```

### Clean local cache

```bash
$ hexo clean
```


### Generate static files

``` bash
$ hexo g
```


### Deploy to remote sites

``` bash
$ hexo d
```
